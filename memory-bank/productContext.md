# Product Context

This file provides a high-level overview of the ArenaClub Watch Marketplace prototype.

## Project Goal
Create a high-fidelity prototype for a job interview case study that mimics ArenaClub.com's exact design language and user experience, adapted specifically for the luxury watch market.

## Key Features
*   **Marketplace Browse Page**: A grid-based interface for browsing watches, featuring "Movement Health" badges (analogous to card grading).
*   **Listing Detail Page**: A deep dive into individual watches, including a 360° "Vault View" and a detailed Movement Health Dashboard.
*   **Trade-In Flow**: A seamless experience for swapping watches, including a vault inventory selector and a real-time swap value calculator.

## Overall Architecture
*   **Frontend**: Next.js (React) with Tailwind CSS for rapid, high-fidelity UI development.
*   **Icons**: Lucide-react for clean, modern iconography.
*   **State Management**: React Hooks (useState/useContext) for prototype-level interactivity.
